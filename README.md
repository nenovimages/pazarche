# Пазарче — безплатни обяви за цяла България

**Пазарче** е сайт за безплатни обяви, който свързва купувачи и продавачи директно — без комисиона, без посредници. Купувай и продавай между съседи из цялата страна.

🌐 **Живо:** [pazarche.vercel.app](https://pazarche.vercel.app)

---

## Какво може

- **Регистрация и вход** с имейл и парола (с потвърждение по имейл)
- **Публикуване на обяви** със снимки, цена, локация и описание
- **Качване на снимки** с автоматична оптимизация (свиване + WebP) и галерия
- **Категории** — имоти, автомобили, техника, мода, дом и градина, за детето, услуги, работа, хоби и спорт, животни
- **Специфични полета по категория** — напр. марка/модел/година/пробег за коли, квадратура/етаж за имоти
- **Търсене и филтри** — по дума, категория, град, цена, разстояние (радиус), специфични полета
- **Търсене по локация** с радиус и сортиране „по близост“
- **Любими** — запазване на обяви
- **Маркиране „продадено“** с филтър за скриване
- **Редактиране и триене** на собствени обяви + изглед „Моите обяви“
- **Вътрешни съобщения** между купувач и продавач, с брояч за непрочетени
- **Профил на продавача** — всички негови обяви, „член от“, рейтинг
- **Рейтинги и отзиви** за продавачи (звезди + коментар) с показване в обявите и картичките
- **Адаптивен дизайн** за компютър и телефон

---

## Технологии

- **Frontend:** React + Vite
- **База данни, автентикация и съхранение:** Supabase (PostgreSQL, Auth, Storage)
- **Оптимизация на изображения:** browser-image-compression (свиване и WebP в браузъра)
- **Икони:** lucide-react
- **Хостинг:** Vercel (автоматичен деплой при push към `main`)

## Локално пускане

```bash
npm install
npm run dev
```

Сайтът се нуждае от Supabase проект. Връзката се конфигурира в `src/supabase.js` (URL и публичен ключ).

## Билд за продукция

```bash
npm run build
```

Готовите файлове се генерират в папка `dist/`.

---

## Деплой

Проектът се хоства на Vercel и се деплойва автоматично при всяка промяна в `main` клона на GitHub. Достатъчно е да качиш променените файлове в GitHub — Vercel пуска нов билд сам.
