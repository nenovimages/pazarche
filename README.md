# Пазарче — безплатни обяви

Готов React + Vite проект. Данните се пазят локално в браузъра (`localStorage`),
така че работи на всеки хостинг без база данни.

## Локално стартиране

Нужен е Node.js (версия 18+).

```bash
npm install
npm run dev
```

Отвори адреса, който излиза в терминала (обикновено http://localhost:5173).

## Качване в Vercel (препоръчано)

1. Създай ново repository в GitHub и качи всички файлове от тази папка.
   (Папките `node_modules` и `dist` се пропускат автоматично заради `.gitignore`.)
2. Влез в https://vercel.com с GitHub акаунта си.
3. Натисни **Add New → Project** и избери repo-то.
4. Vercel сам разпознава Vite. Остави настройките по подразбиране:
   - Build Command: `npm run build`
   - Output Directory: `dist`
5. Натисни **Deploy**. След минута получаваш жив линк (тип `pazarche.vercel.app`).

Всеки следващ `git push` обновява сайта автоматично.

## Важно за данните

Обявите се пазят в браузъра на всеки посетител поотделно (`localStorage`).
Това значи, че всеки вижда само своите обяви — добре е за демо или личен
инструмент. За истински сайт, където всички виждат едни и същи обяви, е нужна
база данни (например Supabase).

## Структура

```
index.html          — входна точка
src/main.jsx        — React стартиране
src/Pazarche.jsx    — цялото приложение
```
